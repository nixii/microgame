
# microgame

Microgame will be a small game engine for retro-esque video games.

It will be usable as a visual editor *and* as a static C library.

## Planned Features
- Pure AABB collisions (no meshes, sorry)
- A small custom coding language for use in the editor
- Have "Resources" for commonly used/reused data, or for preloading data
- A full editor for the engine, coded using itself
